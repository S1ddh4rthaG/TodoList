## Run Locally

Clone the project

```bash
  git clone https://github.com/S1ddh4rthaG/TodoList.git
```

Go to the project directory

```bash
  cd TodoList
```

Install project

```bash
  npm install
```

Start the server

```bash
  npm app.js
```
